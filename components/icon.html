Icon

Icons can be used everywhere, using the data-icon-attribute.
Depending on specific keywords, you can control the icon's position, and whether the icon should be a pseudo-element, a background-image or rendered inline.

    Icon sizes

    The default icon-size is 1.5em, but this is updated through the CSS prop --icon-sz for the various combinations.
    Additionally, the following size keywords can set the size, as in data-icon="sm":

    • xs
    • sm
    • md
    • lg
    • xl
    • xxl

    You can also transform:scale icons, using the scale- prefix, eg. data-icon="scale-xs"
