UI: Components

Asset Handler

The Asset Handler component provides a UI for uploading, tagging, and managing assets (images, videos, documents) via an asset server API.
Demo mode: When using a static JSON file (like data.json used in this example), uploads, deletes, and tag changes are disabled; only asset listing is available.

JSON example


    {
      "name": "test1.jpeg",
      "originalname": "test1.jpeg",
      "path": "assets/12/34/56/123456/test1.jpeg",
      "size": 202300,
      "tags": [
        "front"
      ],
      "createdAt": "2025-03-05T14:26:16.789Z",
      "modifiedAt": "2025-03-05T14:26:16.791Z",
      "uploadDate": "2025-03-05T14:26:16.792Z",
      "type": "image",
      "mimeType": "image/jpeg"
    },
