Asymmetrical

These layouts use fixed patterns where the number of items is predetermined and the layout adapts to a specific arrangement. Unlike columns(), ratios() and non-fixed bento() layouts which repeat patterns, these asymmetrical layouts do not repeat.
If you add more items than the layout can handle, they will be hidden.
