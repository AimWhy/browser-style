UI: Components

Number

<ui-number> is a web component for counting numbers. Set start, end and duration (in milliseconds). Set iteration to -1 for infinite or use any positiive integer for a specific amount. Add an optional suffix and target additional CSS Custom Properties through ui-component::part(number)
