UI: Components

GUI Panel

GUI Panel is a draggable container, using the Popover API. The panel can be freely positioned by dragging its title bar, and automatically saves its position between sessions. It features a built-in color scheme toggle, customizable slots for content and footer, and responds to clicks outside the panel for dismissal. The panel's appearance can be customized using CSS custom properties.
