UI: Components

GUI Panel

GUI Panel is a draggable, resizable container, using the Popover API. The panel can be freely positioned by dragging its title bar, and automatically saves its position between sessions. It features a built-in color scheme toggle, and can be docked in a layout.

This panel is docked by default. You can undock it, and move it around. Resize is top and right. Closing will in this case behave like "re-dock". Default, un-docked position is bottom left.
