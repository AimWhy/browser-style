# CSS Icons

Add attribute `data-css-icon` (preferably to an `<i>`-tag) with one of the allowed values:

- chv-up
- chv-down
- chv-left
- chv-right

- check
- cross
- equals
- minus
- plus

- tri-up
- tri-down
- tri-left
- tri-right

---

## Animate

If used within a `<summary>`-tag, the following icons will animate, when the `open`-attribute is added:

- chv-down
- chv-right
- cross-turn
- equals
- plus
- tri-down
- tri-right
