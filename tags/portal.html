<portal>: The Portal element

Go to chrome://flags and chose Enable under Enable Portals
If you're using a CSP, add 'self' under frame-src
For more information on usage, see the spec.

Dummy-code, using the animation API to transition to the portal:

document.querySelectorAll('portal').forEach(portal => portal.addEventListener('click', event => {
	event.preventDefault();
	portal.style = 'position: fixed; inset: 0; block-size: 1vw; inline-size: 1vh; overflow: hidden;';
	portal.animate([
		{
			height: '100vh',
			width: '100vw'
		}], 
		{
			duration: 500,
			fill: 'forwards'
		}).finished;
		setTimeout(() => { portal.activate() }, 600);
}));
