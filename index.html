Welcome to browser.style

browser.style is a small CSS-framework, with a core.css of just 1KB, minified and brotlified.

It enhances your browsers default styles, so you can use plain HTML and still get great results.

All colors are using color-mix(), mixing-in system colors, so they also looks great in dark mode.

If you need form elements and even more color utility-classes, use base.css, which is less than 3KB, minified and brotlified.
